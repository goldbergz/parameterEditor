# Param Editor Test Task

Компонент для редактирования параметров модели товара.

## Запуск

```bash
npm i
npm run dev
npm test
```

## Что реализовано

- Все параметры из `params` отображаются сразу
- Значения инициализируются из `model.paramValues`
- Все поля редактируемые
- Метод `getModel()` возвращает актуальную структуру `Model`
- Архитектура расширяемая через мапу рендереров типов

## Тесты

✔ Инициализация значений  
✔ Корректная работа `getModel()`

## Расширение

Чтобы добавить новый тип параметра:

1. Расширить `Param["type"]`
2. Добавить рендерер в `renderers`

Компонент при этом менять не потребуется.
